Deer Damage Reporting is a website that allows you to enter new deer damage information within the State of Missouri. There is alway a recall feature that allows you to view and/or edit existing deer damage information. You can recall information by County and/or Year of reporting.

We are using version 3.14 of the ESRI ArcGIS API for JavaScript to create the interface that users will be able to enter and view geospatial data. The data driving the map is being store in MDC's spatial data engine (SDE) and published as a feature service (editable) through Arc for Server. Users can also access the data through ESRI's desktop software (ArcMap) either by connecting to sde or through the rest service. Accessing this layer allows all users to be viewing & editing the same layer, so everyone has access to the most up to date data.

Frequently Asked Questions

What am I looking at?

When you open this website you will see a data entry screen with text boxes, drop-down lists and checkboxes for entering a deer damage report. The page is broken down into 4 basic sections: Locale (Landowner | Contact | Site), Deer Damage, Area Affected and Recommendations. When you complete the data entry please press the green "Save the Data" button on the buttom of the page.

There is also a menu bar feature "Recall Reports" that allows you to retrieve existings reports. These reports can be editing and saved if desired. You retrieve reports by County and/or Year. Only 1 report can be retrieved at a time.

What else can I see?

Within the Deer Damage section of this page is an embedded ArcGis Online Map.

There are 7 basic deer damage sub-types that can be placed on the map as polygon map features:

  • Ag Crop
  • Garden
  • Landscaping
  • Orchard
  • Trees
  • Vegetable/Truck Crop
  • Other

These sub-types will display for all entered reports so that you can see the data in context of other deer damage that has already been reported.

How do I control the map?

The ArcGIS API for JavaScript Map Navigation has elements to allow user interaction with the map.

Users can choose the mouse, keyboard, and sliders for panning and zooming a map, depending on what is enabled in the HTML page. By default, users can do the following:
  • Drag the mouse to pan
  • Mouse Scroll Forward to zoom in
  • Mouse Scroll Backward to zoom out
  • SHIFT + Drag the mouse to zoom in
  • SHIFT + CTRL + Drag the mouse to zoom out
  • SHIFT + Click to recenter
  • Double Click to Center and Zoom in
  • SHIFT + Double Click to Center and Zoom in
  • Use arrow keys to pan
  • Use + key to zoom in a level
  • Use - key to zoom out a level
